Mốc hoàn thành

Variables

Khai báo biến trong Go rất gọn, nhưng phải hiểu ngữ cảnh dùng

var name string = "gopher"
var count int

language := "Go"
isReady := true

`:=` là short variable declaration, cực kỳ phổ biến trong thân hàm. Còn `var` thường dùng khi bạn muốn khai báo tường minh kiểu hoặc khi giá trị zero-value có ý nghĩa. Tư duy này khác với C# ở chỗ zero-value được tận dụng nhiều hơn thay vì luôn khởi tạo đầy đủ.

Functions

Function có thể trả nhiều giá trị, đây là một feature bạn sẽ dùng rất thường xuyên

func divide(a int, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }

    return a / b, nil
}

Multi-return làm thay đổi cách bạn thiết kế API so với C#. Thay vì phải tạo result object nhỏ cho mọi thứ, Go cho phép trả dữ liệu chính và lỗi ngay trong chữ ký hàm. Điều này cực kỳ quan trọng ở các bài sau.

Control Flow

Go chỉ có một loop keyword là `for`

for i := 0; i < 3; i++ {
    fmt.Println(i)
}

for index, value := range numbers {
    fmt.Println(index, value)
}

if total := len(numbers); total == 0 {
    fmt.Println("empty")
}

switch role {
case "admin":
    fmt.Println("full access")
case "user":
    fmt.Println("limited access")
default:
    fmt.Println("unknown role")
}

`if` có thể khai báo biến cục bộ ngay trong điều kiện, và `switch` mặc định không fallthrough. Những chi tiết nhỏ này khiến code Go gọn, ít ceremony hơn nhiều chỗ trong C#.

Defer

`defer` là primitive rất đáng nhớ

file, err := os.Open("data.txt")
if err != nil {
    return err
}
defer file.Close()

Hãy xem `defer` như cách close resource an toàn, gọn và rất “near the acquisition site”. Nó không giống `using` block hoàn toàn, nhưng giải quyết cùng nhóm vấn đề: cleanup tài nguyên đúng lúc.

Điểm cần thuộc lòng Nếu đang học Go từ C#, hãy dành thời gian hiểu thật kỹ `slice`, multi-return và `defer`. Đây là ba thứ làm đổi nhịp code của bạn rõ nhất ở giai đoạn đầu.
